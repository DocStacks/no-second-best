# Game Assets

This folder contains game assets that are served by Vite.

## Required Files:

- `michael-saylor.png` - Michael Saylor portrait image for face overlay (recommended size: 200x250px)

## How to Add Michael Saylor Image:

1. Save your Michael Saylor image as `michael-saylor.png`
2. Place it in this `assets/` folder
3. Commit and push to GitHub
4. The image will be accessible at `/assets/michael-saylor.png` in the game

## CoinGecko Altcoin Images:

The game automatically loads altcoin logos from CoinGecko's CDN, so no additional files are needed for altcoins.
